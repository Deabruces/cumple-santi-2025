# Crear imagen OG para WhatsApp

Para que el sitio se vea bien cuando lo compartas por WhatsApp, necesitas crear una imagen llamada `og-image.jpg` en esta carpeta (`public/`).

## Especificaciones recomendadas:

- **Tamaño**: 1200 x 630 px (ratio 1.91:1)
- **Formato**: JPG o PNG
- **Peso**: Menor a 1MB

## Contenido sugerido para la imagen:

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│     🎂 CUMPLEAÑOS DE SANTIAGO SAUD 🎂              │
│                                                     │
│              ¡10 AÑOS!                              │
│                                                     │
│     📅 19 de Octubre • 16:00                       │
│     📍 Escapology - Parque Araucano                │
│                                                     │
│     🔐 Escape Room + 🍕 Picnic                     │
│                                                     │
│            [Imagen de fondo oscuro]                 │
│         [Con gradientes azul/morado]                │
│                                                     │
└─────────────────────────────────────────────────────┘
```

## Herramientas para crear la imagen:

1. **Canva** (gratis, online)
   - Usa template "Facebook Post" o "Open Graph"
   - Aplica fondo degradado oscuro (azul/morado)
   - Agrega el texto con los detalles del evento

2. **Figma** (gratis, online)
   - Crea un frame de 1200x630px
   - Diseña con fondo oscuro y texto claro

3. **Photoshop / GIMP**
   - Si prefieres usar software de escritorio

## Paleta de colores sugerida:

- Fondo: #1e293b → #374151 (degradado)
- Texto principal: #fbbf24 (dorado/amber)
- Texto secundario: #93c5fd (azul claro)
- Acentos: #8b5cf6 (morado)

## Una vez creada:

1. Guarda la imagen como `og-image.jpg`
2. Colócala en la carpeta `public/`
3. El sitio ya está configurado para usarla automáticamente

## Verificar que funciona:

1. Despliega el sitio
2. Compártelo por WhatsApp
3. Deberías ver la imagen en la preview

O usa herramientas online:
